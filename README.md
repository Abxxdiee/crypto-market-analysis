# Crypto Market Analysis 📊

A Python-based data analysis project tracking Bitcoin (BTC) and Ethereum (ETH) 
price trends over 90 days using live market data.

## What This Project Does
- Fetches live crypto price data from CoinGecko API (no API key required)
- Cleans and structures data using pandas
- Generates price trend visualisations for BTC and ETH
- Exports clean data to CSV for further analysis in Power BI

## Tools Used
| Tool | Purpose |
|------|---------|
| Python | Core programming language |
| pandas | Data manipulation and cleaning |
| matplotlib | Data visualisation |
| requests | API data fetching |
| CoinGecko API | Free crypto market data source |

## Key Findings
- Analysed 90 days of BTC and ETH price movements
- Identified price trend patterns and volatility periods
- Visualised correlation between BTC and ETH price action

## Files
- `crypto_analysis.py` — Main analysis script
- `crypto_data.csv` — Cleaned price data output
- `crypto_trends.png` — Price trend charts
